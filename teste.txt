COMANDOS PARA CONFERIR NA EC2

Entrar na EC2:
ssh ubuntu@32.194.241.171

Ir para o projeto:
cd /var/www/html/hdcevents

Status dos servicos:
sudo systemctl status apache2
sudo systemctl status mysql

Rotas Laravel:
php artisan route:list

Status das migrations:
php artisan migrate:status

Consultar eventos no MySQL:
MYSQL_PWD='COLOCAR_AQUI_O_VALOR_DO_SECRET_DB_PASSWORD' mysql -ularaveluser -h127.0.0.1 hdcevents -e "SELECT id, title, city, date, image, items FROM events;"
